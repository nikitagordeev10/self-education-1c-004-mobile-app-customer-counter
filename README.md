Разработка информационной системы диспетчера по транспорту. Выпускная квалификационная работа. Здесь задействованы знания из разных дисциплин университета. основными являются конфигурация по логистикие и приложения . для синхрониации и изучериня была созана дополнительная конфигурация предприятие. 

**Конфигурация «Предприятие». Создана для изучения работы платформы 1С и не предназначена для использования в реальных условиях.**

### О репозитории
Конфигурация предназначена для управления общей структурой вымышленной компании Она включает учет сотрудников, контроль рабочих процессов, финансовое планирование и ведение складского учета сырья. Благодаря этой конфигурации обеспечивается эффективная организация бизнеса и стратегическое управление ресурсами.
